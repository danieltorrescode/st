st - simple terminal
--------------------

my patched version of st

- `git remote add source git://git.suckless.org/st`

- `git pull --rebase source master`

- `wget https://st.suckless.org/patches/patch.diff`

- `patch -p1 < patch.diff`

- `make`

- `make clean install`
